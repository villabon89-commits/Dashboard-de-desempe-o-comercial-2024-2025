# 📊 Dashboard de desempeño comercial 2024–2025 – Andes Retail Group

Este repositorio contiene la guía estructurada y los resultados del proyecto de **Dashboard de Desempeño Comercial** para **Andes Retail Group**, abarcando el período 2024–2025.

El proyecto se enfoca en el análisis de datos de ventas para comprender el rendimiento comercial, identificar factores clave de crecimiento o decrecimiento, y presentar insights accionables a través de un dashboard interactivo.

## 📂 Contenido del repositorio

- `dashboard_analysis.ipynb`
  → Notebook principal con los pasos para la conexión de datos, preparación, diseño del dashboard, creación de visualizaciones y narrativa de insights.

## 📘 Cómo reproducir el análisis

1. Abre `dashboard_analysis.ipynb`.
2. Ejecuta las celdas en orden, siguiendo las instrucciones de cada paso.
3. El notebook guiará a través del proceso de conexión a los datos (externos o locales) y la construcción del análisis.

- Diseñar y planificar un dashboard efectivo con vistas de Overview y Detalle, definiendo KPIs y tipos de gráficos.
- Construir visualizaciones claras y coherentes para presentar el desempeño comercial.
